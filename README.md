Completion Date: 26th Dec, 2024 

## Tools & Technologies

Language: C++ 

Libraries: WinSock2 (Socket Programming) and Standard Template Library (STL) 

Platform: Windows OS 

IDE: Microsoft Visual Studio 

Web Content: HTML, CSS, and JavaScript 

## How to Run:

Environment: Ensure you are using Windows and have Microsoft Visual Studio installed.


Build: Open the project and compile the source files (main.cpp, TcpListener.cpp, WebServer.cpp).


Execution: Run main.cpp.


Access: Open any browser and enter localhost:8080 or http://[YOUR_IP_ADDRESS]:8080 to see the server in action.
